# Sales Dashboard Analysis

## 📊 Project Overview
An interactive Sales Performance Dashboard built using Excel to analyze 
1,000 orders from the Superstore dataset (Kaggle).

## 🛠️ Tools Used
- Microsoft Excel (Pivot Tables, Slicers, Charts)

## 📌 Key KPIs Tracked
- Total Sales: ₹25,67,962
- Total Orders: 1,000
- Total Profit: ₹4,44,048

## 🔍 Key Insights
- East region had the highest sales
- Furniture, Office Supplies and Technology contributed roughly equally
- Monthly trend analysis revealed peak and low-performing periods
